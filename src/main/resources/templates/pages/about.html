About this project

Application

This application can help you to manage your tasks. Here you can:
Create new task
Add tag to your task
Search your tasks by tag
Edit your tasks
Edit tags in tasks
Delete completed tasks
Also tags in tasks are autoformated. So if you type tag like my new tag then formatted tag will be #my-new-tag .
